# Alpaca - Ask, Done.

<img src="logo.png" alt="Alpaca Logo" width="200"/>

Your private AI assistant that actually gets things done. Chat with AI that can read your emails, schedule meetings, update Notion pages, and connect to thousands of other apps — all while keeping your data on your machine.

## Features

- **Private & Open Source** - Your data stays on your machine. See every line of code.
- **Any AI You Want** - Use free local models with [Ollama](https://ollama.ai), or connect to OpenAI, Anthropic, Google, and any other OpenAI-compatible providers.
- **Thousands of MCP Servers** - Choose from [thousands of MCP servers](https://smithery.ai/servers) for Gmail, Calendar, Notion, Slack, and more — all in one place.
- **No Clutter** - No history. No threads. Just ask, and you're done.

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build application
npm run build

# Package application
cd apps/desktop
npm run package
```

## Requirements

- Node.js >= 18
- npm 10.0.0
